# การจำลองระบบจัดการธนาคาร (Banking System Simulation) 'Pre-coded Not the Final Product'

### Dev By หมีน้อยคอยรัก

โครงงานนี้เป็นการพัฒนาโปรแกรมจำลองการทำงานของระบบธนาคาร โดยใช้ภาษา C++ ร่วมกับ การทำ GUI เพื่อให้ผู้ใช้งานสามารถโต้ตอบกับระบบได้สะดวกและใกล้เคียงกับการใช้งานระบบธนาคารจริงมากขึ้น โปรแกรมมีวัตถุประสงค์เพื่อแสดงให้เห็นถึงการประยุกต์ใช้การเขียนโปรแกรมด้วยภาษา C++

ระบบสามารถแบ่งผู้ใช้งานออกเป็น 2 ประเภทหลัก ได้แก่ ผู้ใช้ทั่วไป (User) และ ผู้ดูแลระบบ (Admin) โดยผู้ใช้ทั่วไปสามารถสมัครสมาชิกเพื่อเปิดบัญชีธนาคารใหม่ เข้าสู่ระบบด้วยชื่อผู้ใช้และรหัสผ่าน และทำธุรกรรมทางการเงินพื้นฐานได้ ส่วนผู้ดูแลระบบสามารถดูแลจัดการข้อมูลบัญชีทั้งหมดภายในระบบ

## ความสามารถหลักของระบบในฐานะของ User มีดังนี้
1. **สมัครสมาชิกและเปิดบัญชีธนาคาร**: ผู้ใช้สามารถกรอกข้อมูลส่วนตัว เช่น ชื่อ เลขบัญชี และรหัสผ่าน ระบบจะบันทึกข้อมูลบัญชีลงในไฟล์เพื่อเก็บไว้ใช้งานในอนาคต โดยเลขบัญชีจะต้องไม่ซ้ำกับบัญชีที่มีอยู่แล้วในระบบ
2. **ระบบเข้าสู่ระบบ (Login System)**: ผู้ใช้ต้องเข้าสู่ระบบด้วยชื่อบัญชีและรหัสผ่านที่ถูกต้อง เพื่อความปลอดภัยและแยกข้อมูลของผู้ใช้แต่ละรายออกจากกัน
3. **การฝากเงิน (Deposit)**: ผู้ใช้สามารถฝากเงินเข้าบัญชีของตนเอง ระบบจะตรวจสอบจำนวนเงินที่ป้อนเข้ามา และปรับยอดเงินคงเหลือ พร้อมทั้งบันทึกประวัติการทำรายการ
4. **การถอนเงิน (Withdraw)**: ผู้ใช้สามารถถอนเงินจากบัญชี โดยระบบจะตรวจสอบว่ายอดเงินคงเหลือเพียงพอหรือไม่ หากไม่เพียงพอ ระบบจะแจ้งเตือนและไม่อนุญาตให้ทำรายการ
5. **การโอนเงิน (Transfer)**: ผู้ใช้สามารถโอนเงินไปยังบัญชีอื่นภายในระบบ โดยต้องระบุเลขบัญชีปลายทางที่ถูกต้อง ระบบจะดำเนินการหักเงินจากบัญชีต้นทางไปเพิ่มเงินให้บัญชีปลายทาง พร้อมบันทึกประวัติการทำธุรกรรมทั้งสองฝั่ง
6. **การแสดงประวัติธุรกรรม (Transaction History)**: ระบบสามารถแสดงรายการฝาก ถอน และโอนเงินย้อนหลัง โดยแสดงข้อมูล เช่น ประเภทธุรกรรม วันที่ เวลา และจำนวนเงิน เพื่อให้ผู้ใช้ตรวจสอบความถูกต้องของการทำรายการ

โดยทั้ง Project จะเขียนด้วยภาษา C++ ในด้าน GUI จะใช้ ระบบหน้าบ้านเป็น Desktop App จะออกแบบหน้าจอให้ใช้งานง่าย เช่น หน้าจอเข้าสู่ระบบ หน้าจอเมนูหลัก และหน้าจอทำธุรกรรมต่าง ๆ โดยใช้ปุ่ม ช่องกรอกข้อมูล และตารางแสดงข้อมูล รวมทั้งระบบหลังบ้าน (เขียนโค้ดจัดการไฟล์ txt ด้วย ภาษา C++)
